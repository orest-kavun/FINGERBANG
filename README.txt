FINGERBANG—A fundamentally new programming language which uses emoji instead of boring letters.

👉—Increment the data pointer;
👈—Decrement the data pointer;
👆—Increment the byte at the data pointer;
👇—Decrement the byte at the data ponter;
👍—If the byte at the data pointer is zero, then jump to the command after the matching 👎;
👎—If the byte at the data pointer is nonzero, then jump to the command after the matching 👍;
🤮—Output the data at the data ponter as an ASCII character;
🧠—Print all memory content as ASCII characters;

    (Maybe) modify settings.c
    Just run make.

How to use:
    ./fingerbang < test.fb
